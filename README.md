# 📊 StratoIQ

**StratoIQ** is a high-performance product insights platform engineered to give Product Managers deep, visual clarity into user journeys, friction points, and feature effectiveness. From onboarding flows to A/B test performance, StratoIQ delivers end-to-end behavioral intelligence to guide data-backed product decisions.

---

## 1. Features

* ⚖ Behavior funnels with real-time segmentation
* 📊 Heatmaps to analyze user interaction hotspots
* 🌐 A/B test tracking for experiment validation
* 🔢 Drop-off detection in multi-step journeys
* 🤖 Visual analytics dashboard for PMs and teams

---

## 2. 🛡️ Core Security Features

* Encrypted data storage using AES-256
* JWT-based secure authentication
* Docker container isolation across services
* Role-Based Access Control (RBAC)
